## What it does
Audits your exported browser password CSV locally. 
Nothing leaves your machine. Checks for reused passwords, 
weak patterns, common passwords, and personal info in passwords.
Sorts worst passwords first.

## How to use
1. Export passwords from Chrome: chrome://password-manager/passwords → Settings → Export
2. Run: python password_auditor.py
3. Select your CSV when the file picker opens
4. Enter a name for your report file
5. Open the report to see your results
